k8s环境搭建

发表于 2021-03-13 | 分类于 运维
字数统计: 1.8k | 阅读时长 ≈ 8

1. 操作系统

  • 操作系统：centos7.6

    下载地址：http://mirrors.sohu.com/centos/7.6.1810/isos/x86_64/CentOS-7-x86_64-DVD-1810.iso

  • 系统安装盘制作

    采用 UltraISO 制作系统盘，下载地址 https://www.onlinedown.net/soft/614.htm

    制作以及安装流程：https://www.cnblogs.com/zhuimengdeyuanyuan/p/13440561.html

2. 环境准备

可以使用物理服务器，没有也可用虚拟机代替，需要设置处理器>=2，关于 VMWare 虚拟机及设置静态 ip 可以参考文章 https://blog.csdn.net/tiandixuanwuliang/article/details/81283316

开始环境准备，需要在所有节点进行下面的操作

  • 设置主机名 hostname，管理节点设置主机名为 master

需要设置其他主机名称时，可将 master 替换为正确的主机名 node1、node2 即可

  • 编辑 /etc/hosts 文件，添加解析

如果有多个节点，把其他的 ip hostname 加上，每台机器上都如此

  • 关闭防火墙、selinux 和 swap

  • 配置内核参数，将桥接的 IPv4 流量传递到 iptables 的链

  • 配置国内 yum 源
  • 配置国内 Kubernetes 源

  • 配置 docker 源

3. 软件安装

需要在所有节点进行下面的操作

  • 安装 docker

  • 安装 kubeadm、kubelet、kubectl

Kubelet 负责与其他节点集群通信，并进行本节点 Pod 和容器生命周期的管理。Kubeadm 是 Kubernetes 的自动化部署工具，降低了部署难度，提高效率。Kubectl 是 Kubernetes 集群管理工具
